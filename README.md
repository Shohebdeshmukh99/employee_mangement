#  My Flutter App

A simple Flutter app to display and search employee details with a modern UI.

## 📦 Features
- Fetch employees from API
- Search by name
- View employee details in bottom sheet

## 🛠️ Tech Stack
- Flutter
- Dart
- HTTP package
